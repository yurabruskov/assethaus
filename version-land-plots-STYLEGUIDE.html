Фирменные элементы

3D Logo Element

Объемный логотип в виде куба с пульсирующими гранями. Используется в заставках и промо-материалах.

Building Block

Базовый элемент — здание с solid стенами и линейными краями. Символизирует недвижимость.

Data Grid

TRON-style сетка с cyan линиями. Представляет технологичность и структурированность данных.

Value Pyramid

Пирамида из блоков с gradient от красного к зеленому. Визуализирует рост стоимости активов.

Token Ring

Вращающееся кольцо из маленьких кубиков. Символизирует токенизацию и фрагментацию активов.

Network Nodes

Связанные между собой узлы-кубики. Представляет блокчейн и децентрализованную сеть.
