ELEMENT 02

Building Block

Базовый фирменный элемент — объемное здание с solid стенами и wireframe гранями. Символизирует недвижимость как физический актив. Используется цветовая палитра от темных к ярким оттенкам в зависимости от ценности актива. Solid стены дают объем, а EdgesGeometry создает характерную TRON-эстетику.

Geometry
BoxGeometry + EdgesGeometry
Material
MeshBasicMaterial + LineBasicMaterial
Color Range
#326B8C → #FF6B6B
Animation
Gentle height pulse + Fade-in
Use Case
Asset visualization, property maps
Opacity
Solid walls (1.0) + Edges (0.0-0.8)
ELEMENT 03

TRON-Style Grid

Характерная сетка в стиле TRON с cyan центральными линиями и темно-синими grid линиями. Создает ощущение технологичности, цифровизации и структурированности данных. Используется как ground plane во всех 3D визуализациях. Полупрозрачность (opacity 0.3-0.5) позволяет фокусироваться на основных элементах.

06a. 3D Фирменные элементы

Уникальные трехмерные объекты в стиле Land Plots 3D визуализации. Эти элементы создают узнаваемую визуальную идентичность и используются в интерактивных демонстрациях, презентациях и промо-материалах.

3D ELEMENT 01

Building Block — Высотное здание

Ключевой фирменный элемент: объемное здание переменной высоты с solid стенами цвета value-gradient (#326B8C → #FF6B6B) и wireframe гранями. Используется для визуализации недвижимых активов, портфелей и карт стоимости.

Building Block Interactive
Geometry
BoxGeometry (variable height)
Materials
Solid walls + EdgesGeometry
Color Formula
R=255*(h/25), G=107, B=107+(1-h/25)*80
Animation
Gentle pulse, rotation on hover
3D ELEMENT 02

Park Block — Зеленая зона

Низкое плоское здание зеленого цвета (#6BFF8A) представляет парки, зеленые зоны и экологические элементы в визуализациях. Отличается малой высотой (0.5-1.5 units) и яркостью, создавая контраст с value buildings.

Анимированная сетка в стиле TRON с cyan (#00D4FF) центральными линиями и темно-синими (#003366) grid линиями. Создает digital/cyber эстетику. Используется как ground plane во всех 3D сценах.

3D ELEMENT 04

Value Pyramid — Пирамида роста

Пирамидальная структура из кубов с градиентом от низких значений (темные, внизу) к высоким (яркие красные, вверху). Визуализирует рост стоимости, иерархию активов и инвестиционные уровни.

Маленький вращающийся куб с wireframe эстетикой в цветах бренда (#6BFF8A, #00D4FF, #FF6B6B). Используется для визуализации токенов, фрагментов активов, частиц данных. Часто в группах или orbital rings.

Преобразуем физические активы в цифровые токены на блокчейне. Каждый токен представляет долю собственности и дает право на доход от аренды.

Прозрачность
