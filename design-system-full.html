Abstract Forms

Five core concepts, each represented by unique distorted geometry. Every page load generates different variations.

The Foundation
Irregular polygon (5-8 vertices, random). Represents first investment. Solid, glowing, unique every time.
Use: Empty states, onboarding
The Cluster
3-5 organic forms orbiting. Represents portfolio diversification. Each form unique, colors gradient.
Use: Portfolio overview
The Flow
Flowing wave surface from irregular polygons. Represents market liquidity. Vertex displacement creates motion.
Use: Liquidity, marketplace
The Network
Connected nodes (spheres + lines). Random positions, pulsing nodes. Represents ecosystem connections.
Use: Partnerships, ecosystem
The Skyline
7-12 distorted vertical forms with ascending heights. Color-coded by value. Represents growth.
Use: Analytics, growth charts

Code Examples

Create unique abstract forms with controlled randomness.

Generate Abstract Form

// Create unique distorted polygon function createAbstractForm(seed) { const vertices = 5 + Math.floor(seededRandom(seed) * 8); const color = COLORS[Math.floor(seededRandom(seed + 1) * COLORS.length)]; const shape = new THREE.Shape(); for (let i = 0; i < vertices; i++) { const angle = (Math.PI * 2 / vertices) * i; const radius = 1.5 * (0.8 + seededRandom(seed + i) * 0.4); const x = Math.cos(angle) * radius; const y = Math.sin(angle) * radius; if (i === 0) { shape.moveTo(x, y); } else { const prevAngle = (Math.PI * 2 / vertices) * (i - 1); const cpx = Math.cos((angle + prevAngle) / 2) * radius * 1.2; const cpy = Math.sin((angle + prevAngle) / 2) * radius * 1.2; shape.bezierCurveTo(cpx, cpy, cpx, cpy, x, y); } } const geometry = new THREE.ExtrudeGeometry(shape, { depth: 0.5 + seededRandom(seed + 99) * 1.5, bevelEnabled: true, bevelThickness: 0.1 }); const material = new THREE.MeshStandardMaterial({ color: color, metalness: 0.3, roughness: 0.7 }); return new THREE.Mesh(geometry, material); }
