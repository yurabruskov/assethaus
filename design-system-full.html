Design Philosophy

We don't show literal buildings. We show abstract cities where each structure represents a tokenized asset. Height shows value, color shows performance, position shows diversification.

Why Cities?
Real estate portfolios are like cities ‚Äî diverse, growing, interconnected. A single property is a building. A portfolio is an entire urban landscape. This metaphor is intuitive and powerful.
Why Abstract?
Abstraction is universal. It works for any property type, any location, any market. It's timeless, scalable, and distinctly modern.

Abstract Forms

Five core 3D compositions representing key concepts in real estate investment. Each uses the city metaphor in unique ways.

The Foundation
A single solid block represents the foundation of your portfolio ‚Äî your first investment, stable and strong.
Use: Empty states, onboarding
The Cluster
Multiple buildings of varying heights show diversification ‚Äî different asset types, different values, balanced portfolio.
Use: Portfolio overview
The Skyline
Buildings arranged by height create an ascending skyline ‚Äî visual representation of portfolio growth over time.
Use: Growth charts, success states
The Grid
Organized grid of uniform buildings represents market structure ‚Äî orderly, accessible, liquid markets.
Use: Marketplace, available assets
The District
Circular arrangement of buildings around a central plaza ‚Äî represents market cycles and ecosystem connectivity.
Use: Analytics, market overview

Code Examples

Ready-to-use Three.js code for creating portfolio cityscapes.

Create a Single Building

// Create a building (asset) const height = assetValue / 1000000; // $1M = 1 unit height const color = performance > 0 ? 0x00ff00 : 0xff0000; const geometry = new THREE.BoxGeometry(1, height, 1); const material = new THREE.MeshStandardMaterial({ color: color, metalness: 0.3, roughness: 0.7 }); const building = new THREE.Mesh(geometry, material); building.position.y = height / 2; scene.add(building);

Create a City (Portfolio)

// Create portfolio city const gridSize = 12; const spacing = 1.5; for (let x = 0; x < gridSize; x++) { for (let z = 0; z < gridSize; z++) { const asset = portfolio[x * gridSize + z]; if (!asset) continue; const height = asset.value / 1000000; const building = createBuilding(height, asset.performance); building.position.x = (x - gridSize/2) * spacing; building.position.z = (z - gridSize/2) * spacing; scene.add(building); } }
